## Number Guessing Game
This is my first Python project.

## Description
It is a simple number guessing game where the computer chooses a random number, and the player tries to guess it.

## Features
- Randon Number Generation 
- User input
- High/Low hints
- Unlimited attempts
- Winning message

## How to Run
```bash
python main.py
```
